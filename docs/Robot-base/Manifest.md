
# Manifest

The `Manifest.java` file is a crucial component of the Robot-base project, serving as a central registry for all the modules and components that make up the robot's functionality. It provides a structured way to manage and organize the various parts of the robot, ensuring that they can be easily accessed and utilized throughout the codebase. Additionally, it contains the `RunMode`which defines the current mode of the robot (real or simulation). Furthermore, it has a list of all the coded components such as mechanisms or telemetry as boolean variables, allowing teams to easily enable or disable them if their robot's components change during the competition.
